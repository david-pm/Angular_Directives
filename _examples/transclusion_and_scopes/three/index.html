Transclusion and Scopes

Transcluded html (cloned and pulled into directive's template) does not have access to the isolated scope of the directive (except header since its declared on the directive's scope object).

Transcluded html is also not a new child scope of the directive, but rather a child scope of the controller.

The link function has an optional fifth parameter called the transclude function. The transclude function receives an optional first parameter, and a another function. This other function clones the transcluded html and allows you to do whatever you need to do with it. The first parameter takes in a scope, which will be used as the scope that will be attached to the cloned html.

Hello, I am {{person.name}} and,

I am a {{person.profession}}
