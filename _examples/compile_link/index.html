Grasping controller, link, and compile functions in directives

This variable comes from the view's main controller: {{job}}

Mess with the directive's scopes and see how things are affected. Good rule's of thumb:

  • Controller is used when you need to share or re-use directive methods.
  • Link is used when you need to modify the DOM, attach event handlers or run some jQuery code.
  • Compile is used when you need to perform some operations before the link function is run (rare use case). Compile used to include the transclude function, but that now exists in the link function reducing compile's utility.
