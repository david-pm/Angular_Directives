Mess with the directive's scopes and see how things are affected. Good rule's of thumb:

  • Compile is used to modify directive templates, append Angular expressions, inject another directive to the parent directive, etc.
  • Controller is used when you need to share or reuse directive method.
  • Link is used when you need to modify DOM, attach event handler or run some jQuery code.
